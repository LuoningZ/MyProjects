Project Title

Mobile App Design


"Plant" App Design


This self management app allow users to plant a tree in a real world as a result of their own accomplished goal. Users can set their resolutions (goals) as projects consists of smaller tasks for each accomplished they are going to be awarded with positive feedback. This positive feedback is going to be shown through animation. Each project will be given a seed/sapling at the start. Every each task accomplished within a project is going to be awarded in form of some plant grow supplement (sunshine, water, fertilizer). If the user manages to finish hers or his project, a real tree is going to be planted! App provides users with predefined library from which user can select between most common resolutions/projects and then customize them as they prefer. This app will allow users to record every each step towards their goal. It is designed as a self promotion diary allowing users to review their achievements.
